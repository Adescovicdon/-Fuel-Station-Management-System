# -Fuel-Station-Management-System
A web-based fuel station management platform designed to digitize daily operations, sales, and inventory tracking for petroleum companies. This system replaces manual record-keeping with a modern, efficient, and secure solution that improves accuracy and saves time.
🚀 Features

🔑 Multi-user authentication (Director, Manager, Cashier, Attendant)

🛢️ Tank & Pump Monitoring – track stock levels, dips, and variances

💰 Daily Sales Recording – log and review cash/credit sales per product

📊 Reports & Analytics – auto-generate sales summaries and variance reports

📦 Inventory Management – manage fuel supply, deliveries, and consumption

📝 Audit & History – keep a record of all transactions for accountability
🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript, Bootstrap / Tailwind

Backend: PHP (Laravel / Core PHP)

Database: MySQL

Version Control: Git & GitHub

⚙️ Setup Instructions

Clone the repository

git clone https://github.com/Adescovicdon/-Fuel-Station-Management-System.git
cd Fuel-Station-Management-System


Import the database (petroleum.sql) into MySQL

Configure your .env / config.php with DB credentials

Run on local server (XAMPP, WAMP, Laragon, etc.)

Login with default credentials (director/director)
